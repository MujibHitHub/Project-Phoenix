JavaScript must be enabled. Please follow the instructions below to turn them on

If you are using Internet Explorer:
1. On web browser menu click "Tools" menu and select "Internet Options".
2. In the "Internet Options" window select the "Security" tab.
3. On the "Security" tab click on the "Custom level..." button.
4. When the "Security Settings - Internet Zone" dialog window opens, look for the "Scripting" section.
5. In the "Active Scripting" item select "Enable".
6. When the "Warning!" window pops out asking "Are you sure you want to change the settings for this zone?" select "Yes".
7. In the "Internet Options" window click on the "OK" button to close it.
8. Paste the url of the test again in the address bar and hit enter

If you are using Google Chrome:
1. On the web browser menu click on "Settings".
2. In the "Settings" section click on the "Show advanced settings..."
3. On the "Security" tab click on the "Custom level..." button.
4. Under the the "Privacy" click on the "Content settings..."
5. When the dialog window opens, look for the "JavaScript" section and select "Allow all sites to run JavaScript (recommended)".
6. Click on the "OK" button to close it.
7. Paste the url of the test again in the address bar and hit enter

If you are using Mozilla Firefox:
1. In the address bar, type about:config and press Enter.
2. Click "I'll be careful, I promise" if a warning message appears.
3. In the search box, search for javascript.enabled
4. Toggle the "javascript.enabled" preference (right-click and select "Toggle" or double-click the preference) to change the value from "false" to "true". 5. Paste the url of the test again in the address bar and hit enter

If you are using Apple Safari:
1. On the web browser menu click on "Preferences".
2. In the "Preferences" window select the "Security" tab.
3. In the "Security" tab section "Web content" mark the "Enable JavaScript" checkbox.
4. Paste the url of the test again in the address bar and hit enter

If you are using iPad Safari:
1. Tap Settings > Safari > Advanced and turn JavaScript on
2. Paste the url of the test again in the address bar and hit enter

If you are using Android browser:
1. Tap web browser Settings > Content settings > Check the "Turn on JavaScript"
2. Paste the url of the test again in the address bar and hit enter
